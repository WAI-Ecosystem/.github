# Security Policy

## Supported Repositories
This policy applies to repositories under the WAI-Ecosystem organization unless a repository has its own SECURITY.md.

## Reporting a Vulnerability
Please do not report security vulnerabilities through public GitHub issues.

Report security concerns privately to:
- Email: security@waiecosystem.com

Please include:
- affected repository
- short description of the issue
- reproduction steps
- possible impact
- screenshots or logs if available

## Response
We will review reports as soon as possible and coordinate any necessary fixes before public disclosure.
